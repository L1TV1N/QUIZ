Откройте файл backend/.env и замените:
```bash

**Windows:**

  # Терминал 2
  cd frontend && npm install && npm run dev
  ```

→ Откройте браузер: http://localhost:5173

## 🔧 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### GigaChat ошибка (429, 401, etc)
✅ Автоматически включится fallback (красивые шаблоны)
✅ Все функции работают, только без реального AI
→ Проверьте интернет соединение

### Telegram не отправляется
→ Проверьте Chat ID
→ Убедитесь что бот активен
→ Смотрите лог бэкенда (Ctrl+C и看看 последние строки)

### Frontend не загружается
→ Проверьте что npm install успешно установил зависимости
→ Удалите node_modules и переустановите: rm -rf node_modules && npm install

### Database не создаётся
→ Проверьте права доступа на папку backend
→ Удалите backend/database.db если он повреждён
→ Перезагрузите приложение
