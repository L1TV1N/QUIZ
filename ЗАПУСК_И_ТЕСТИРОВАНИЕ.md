# 🚀 Smart Quiz - Запуск и Тестирование

## ✅ Что уже работает

### Backend (Flask)
- ✅ **Статус**: Запущен на `http://localhost:5000`
- ✅ **API**: Доступен на `http://localhost:5000/api`

## 📋 Текущий статус

### Реализованные компоненты
1. ✅ Zustand store с полной структурой данных
2. ✅ 9-step квиз с поддержкой двух режимов:
   - Обычный режим (6 шагов)
   - AI-режим (9 шагов с дополнительными AI вопросами)
3. ✅ Все UI компоненты:
   - Step0ModeSelection (выбор режима)
   - Step1Intro (введение)
   - Step2Property (тип помещения, 6 вариантов)
   - Step3Zones (множественный выбор зон, 10 вариантов)
   - Step4Area (ползунок площади 20-300 м²)
   - Step4Style (7 стилей, одиночный выбор)
   - Step5Budget (5 бюджетов, одиночный выбор)
   - Step6Contacts (форма контактов)
   - Step7AiQuestions (AI вопросы для AI-режима)
   - ResultScreen (финальный экран)

4. ✅ Backend API:
   - Модель Lead с полной структурой данных
   - Валидация входных данных (телефон ≥ 10 цифр)
   - Сохранение в БД (SQLite)
   - Отправка в Telegram Bot
   - Поддержка AI режима с fallback на обычный текст

## ⚠️ Что требует установки

### Frontend (React + Vite)
Требуется установить Node.js v18+ и npm

**Шаг 1: Установите Node.js**
- Windows: Перейдите на https://nodejs.org/
- Скачайте LTS версию (v18 или выше)
- Запустите инсталлер через мастер установки

**Шаг 2: Проверьте установку**
```bash
node --version
npm --version
```

**Шаг 3: Установите зависимости**
```bash
cd frontend
npm install
```

**Шаг 4: Запустите dev сервер**
```bash
npm run dev
```

Frontend будет доступен на `http://localhost:5173`

## 🧪 Тестирование

### Способ 1: Через браузер (рекомендуется)
1. Откройте `http://localhost:8000/test_api.html` в браузере
2. Нажмите "Проверить статус"
3. Заполните тестовые данные
4. Нажмите "Отправить на backend"
5. Проверьте ответ от API

### Способ 2: Через Python скрипт
```bash
cd project_root
python test_api_direct.py
```

### Способ 3: Через cURL (если установлен ZGit Bash)
```bash
curl -X GET http://localhost:5000/api/health
curl -X POST http://localhost:5000/api/submit-quiz \
  -H "Content-Type: application/json" \
  -d '{"aiMode":false,"propertyType":"apartment",...}'
```

## 🐛 Troubleshooting

**Проблема**: Backend не запускается
- Проверьте: `python -m pip list` (должны быть Flask, flask-cors, SQLAlchemy)
- Переустановите: `pip install -r backend/requirements.txt`

**Проблема**: Frontend не запускается
- Проверьте: `node --version` и `npm --version` должны быть установлены
- Переустановите зависимости: `cd frontend && rm -rf node_modules && npm install`

**Проблема**: CORS ошибки
- Backend уже настроен с `flask-cors`, не должно быть проблем
- Убедитесь что используете правильные URLs (localhost:5173 для frontend, localhost:5000 для backend)
