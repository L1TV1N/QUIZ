💾 ДЛЯ РАЗРАБОТЧИКОВ, КОТОРЫЕ КЛОНИРУЮТ ПРОЕКТ:
================================================================================

git clone https://github.com/YOUR_USERNAME/smart-interior-quiz.git
cd smart-interior-quiz
python -m venv .venv
.venv\Scripts\activate
cd backend
pip install -r requirements.txt

Затем создать backend/.env из env.example и добавить переменные окружения.

================================================================================
🔑 ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ (.env):
================================================================================

TELEGRAM_BOT_TOKEN=your_token
TELEGRAM_CHAT_ID=your_chat_id
GIGACHAT_CREDENTIALS=your_credentials
GIGACHAT_SCOPE=GIGACHAT_API_CORP
GIGACHAT_MODEL=GigaChat-2
GIGACHAT_VERIFY_SSL_CERTS=False
DATABASE_URL=sqlite:///database.db
